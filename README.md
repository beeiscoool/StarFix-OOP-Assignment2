# StarFix – OOP Assignment

StarFix is a console-based space survival quiz game developed using C# and Object-Oriented Programming (OOP) concepts.

The player takes on the role of a spaceship pilot trying to stabilize their ship by answering questions correctly. Each correct answer helps repair the ship, while wrong answers bring the ship closer to failure.

---

## Features

- Interactive quiz-based gameplay  
- Randomized question order (shuffle feature)  
- Score tracking system  
- Ship status feedback (stabilized, critical, etc.)  
- Simple and clean console interface  

---

## Concepts Used

This project demonstrates the use of:

- Classes and Objects  
- Encapsulation  
- Methods and Functions  
- Lists / Arrays  
- Randomization (Shuffle)  
- Conditional Statements  
- Loops  

---

## How to Play

1. Run the program  
2. Answer the questions displayed  
3. Each correct answer improves ship stability  
4. Try to survive until all questions are completed
